ÉXODO.
37.
Huan teipa Bezaleel quisencajqui nopa caxa ica cuahuitl acacia huan quipixqui 110 centímetros ihuehueyaca, huan 65 centímetros ipatlajca, huan 65 centímetros ihuejcapanca. 
Iijtic huan ipani quixajqui ica senquistoc oro, huan yahualtic quipixtoya itejteno tlen oro. 
Huan quichihuili nahui anillos tlen oro huan quitlalili najcac ipan inacastla. 
Nojquiya quisencajqui cuahuitl huehueyac huan quixajqui ica oro. 
Huan ipan nopa anillos quicalaqui nopa cuahuitl huehueyac para quej nopa huelis quihuicase nopa caxa. 
Teipa quichijqui itzajca nopa caxa campa Toteco quintlasojtlas masehualme. 
Nojquiya iixco itzajca quisencahuili ome itequihuajcahua Toteco catli inintoca querubines catli tlatejtzontli ica martillo. Huan nopa tequihuajme huan itzajca nopa caxa elqui san se tlatectli oro. Huan nopa tequihuajme moquetztoyaj ipan sese iteno itzajca. 
Se itequihua Toteco moquetztoc ipan se iteno huan ne seyoc ipan seyoc iteno. Nopa querubines huan itzajca nopa caxa elqui san se tlatectli oro tlatejtzontli. 
Nopa tequihuajme moixnamictoyaj huan moeltlapal tlanantoyaj quen quiecahuiliyayaj huan quitlachiliyayaj itzajca nopa caxa campa TOTECO quintlasojtlas masehualme. 
Nojquiya quisencajqui nopa cuamesa ica cuahuitl acacia. Huan quipixtoya ihuehueyaca 90 centímetros, huan 45 centímetros ipatlajca, huan 65 centímetros ihuejcapanca. 
Huan quixajqui ica senquistoc oro. Huan quichihuilijque nopa cuamesa se icorona tlen oro catli quitenyahualo iixco. 
Nojquiya quitenyahualo iixco nopa cuamesa ica se pilhuapaltzi catli quipiya san chicome centímetros ipatlajca. Huan yahualtic quitejtequi para quichihuili iyejyejca huan quixajqui nochi ica senquistoc oro. 
Huan quichihuili nahui anillos tlen oro, huan quitlali najcac ipan inacastla nopa cuamesa campa pehua iicxihua iquespa nopa mesa. 
Nopa anillos eltoya huejcapa ipan iicxi nechca nopa pilhuapaltzi catli quitentzajqui nopa cuamesa. Huan quisencajqui cuahuitl huehueyac huan quicalaqui ipan nopa anillos ipan nopa cuamesa para quej nopa huelis quihuicase. 
Huan para quichihuas nopa cuahuitl huehueyac para ica quihuicase nopa cuamesa quitequihui cuahuitl acacia, huan quixajqui ica oro. 
Nojquiya quisencajqui catli quitequihuise ipan cuamesa quen platome, huejhueyi cucharas, teposhuitzcucharas huan huejhueyi xarros para tlacajcahualistli catli atic. 
Nojquiya quisencajqui se tlahuil quetzali catli senquistoc oro. Iicxi, icuayo, ipilcaxitzitzi, ixochiyo mimilijtoc huan cahuantoc huan nochi iyejca quisencahuili ica san se tlatectli oro catli tlatejtzonali ica martillo. 
Huan quipixtoya chicuase imacuayohua; eyi imacuayohua ipan se lado, huan ne se eyi ipan seyoc lado. 
Ipan sese imacuayo quipixqui eyi pilcaxitzitzi catli ipan tlatlatise huan quipixqui itlachiyalis quen se almendra ixochiyo catli mimilijtoc huan se catli cahuantoc. Huan ipan nochi nopa chicuase imacuayohua quej nopa eliyaya. 
Huan ipan icuayo nopa tlahuil quetzali quipiyayaya nahui pilcaxitzitzi, sese ica itlachiyalis se xochitl almendra, se nesiyaya se xochitl mimilijtoc huan se quen se xochitl cahuantoc. 
Huan itzintla tlen sesen nopa chicuase imacuayohua catli tlatzquitoyaj ipan icuayo nopa tlahuil quetzali quipiyayaya se xochitl cahuantoc. 
Huan nochi ixochiyohua huan imacuayohua huan icuayo eliyaya san se tlatectli oro huan tlatejtejtzontli ica martillo. 
Nojquiya quichihuili chicome piltlahuiltzitzi, huan nopa cucharas catli ica quisehuijque nopa tlahuilme, huan quichihuilijque iplatohua. Huan nochi quisencajque ica senquistoc oro. 
Huan nopa tlahuil quetzali tlen oro quipixqui 33 kilos ietica. 
Nojquiya quichijqui se tlaixpamitl campa quitlatise copali huan elqui cuadradojtic, ica 45 centímetros ihuehueyaca huan ipatlajca, huan quipiyayaya icuacua ipan nochi nahui inacastla, huan nochi san eliyaya se tlatectli cuahuitl. 
Huan quixajqui ica senquistoc oro nochi itzontzajca, huan iquespa, huan icuacua. Huan yahualtic iixco quiquetzqui se icorona tlen oro catli quiyectlali. 
Huan nojquiya quichihuili nahui anillos itlaj nopa nahui iesquinajyo, huan nopona quicalaqui nopa cuahuitl huehueyac para ica quihuicase. 
Huan nopa cuahuitl huehueyac quisencajqui ica cuahuitl acacia, huan quixajqui ica oro. 
Nojquiya quisencajqui nopa aceite catli tlatzejtzeloltic para tlaiyocatlalise para Toteco, huan quisencajqui nopa copali catli ajhuiyac mijyotía. Nochi quisencajqui Bezaleel, se tlaajhuiyacasencajquetl catli achi más quimatqui.
