PROVERBIOS.
29.
Aqui moyoltetilía huan amo quichihuilía cuenta quema sequinoc quitlacahualtíaj, tlatocmiquis huan ayacmo oncas pajtli para momanahuis. 
Quema tlanahuatíaj nopa masehualme catli xitlahuaque, nochi masehualme yolpaquij, pero quema tlanahuatíaj nopa masehualme catli fiero inintlalnamiquilis, nopa altepetl ehuani tlaijiyohuíaj. 
Se telpocatl catli mojmostla quinequi elis más tlalnamiqui quipaquilismacas itata, pero aqui nemi campa momecatijca sihuame quipinahualtía itata huan quipolos nochi catli quicahuilijtosquía. 
Se tlanahuatijquetl catli tetlajtolsencahua xitlahuac quichihua para nochi ipan itlal ma eli temachtli. Pero se tequitiquetl catli quiselis tomi ichtacatzi para quichihuas catli amo xitlahuac quitlatzintlahuetziltía itlal. 
Se catli quihueyimati seyoc ica miyac camanali catli yejyectzi para quicajcayahuas, eltoc quen quitlalilía se tlaquetztli iojtipa para ma ipan huetzis. 
Se tlacatl catli amo cuali moijilpía ipan itlajtlacolhua, pero catli nemi xitlahuac, amo. Yaya huicatinemi huan nelía paqui. 
Cuajcualme tlamocuitlahuíaj para catli teicneltzitzi ma quipiyaca nochi caquihuili catli nopa tlanahuatili quiijtohua para quipiyaj, pero se masehuali catli amo quitlepanita TOTECO amo quema mocuesohua por catli teicneltzitzi. 
Catli tehuihuiitaj hueli quinyolajcomanaj nochime ipan se altepetl, pero masehualme catli tlalnamiquij quichihuaj campica para quiseseltíaj cualantli. 
Quema se catli tlalnamiqui quihuica se catli huihuitic iixpa se juez, nopa huihui tlacatl san quihuihuiitas huan quihuetzquilis. Amo tleno quisencahuas para moyoltlalise san sejco. 
Tlacame catli temictíaj quincualancaitaj catli monejnemiltíaj xitlahuac, pero cuajcualme quitemohuaj catli elis cuali para nochi, masque amo cuajcualme. 
Se catli huihuitic cuatzajtzi huan quinextía nochi icualancayo, pero se catli tlalnamiqui moijiyohuía huan moyoltlalía. 
Sintla se tlanahuatijquetl quinchihuilía cuenta masehualme catli istlacatij, nochi itequipanojcahua mocuepase amo cuajcualme. 
Se teicneltzi catli quitlaijiyohuiltíaj huan se catli tetlaijiyohuiltía nochi ome quipiyaj se tlamantli catli san se. Itztoc yaya TOTECO catli quinmacatoc nochi ome nemilistli huan tlachiyalistli. 
Nopa tlanahuatijquetl catli quintlajtolsencahua xitlahuac catli teicneltzitzi quichihuas para itequiticayo ma huejcahuas para miyac xihuitl. 
Se conetl catli quitlacahualtíaj huan quihuitequij ica cuapipitztli quema quinamiqui, teipa elis tlalnamiqui. Pero se conetl catli san quicahuilíaj para ma quichihua catli fiero quinequi, quinpinahualtis itatahua. 
Quema quipiyaj tequiticayotl miyac tlaixpanoca masehualme, momiyaquilía tlajtlacoli, pero masehualme catli monejnemiltía xitlahuac mocahuase para quinitase quema tlamis inintequiticayo. 
Xijxitlahua mocone huan huelis timosiyajquetzas huan amo timoyoltequipachos por ya. Quena, yaya quichihuas para nelía tiyolpaquis. 
Ipan se tlali campa amo aqui quiiximati TOTECO, niyon amo aqui tematiltía icamanal, nopa masehualme quichihuaj miyac tlamantli catli fiero. Pero ipan se tlali campa masehualme quiiximatij huan quitlepanitaj itlanahuatilhua TOTECO, onca hueyi paquilistli. 
Se tequipanojquetl catli amo quitlacaquilía san camanali, monequi tijtlalnamictis huan tijtlatzacuiltis se quentzi. Masque quimachilía icamanal iteco, quemantic amo quichihuilía cuenta catli quinahuatía. 
¿Tiquitztoc se masehuali catli camanalti miyac huan amo quichihuilía cuenta catli quiijtohua? Más cuali quisas se tlacatl huihuitic huan amo ya. 
Sintla tlahuel tiquicnelis se motequipanojca hasta quema noja tziquitetzi, teipa elis quen motelpoca catli quiselis catli tijcahuilijtehuas. 
Se catli yolquentzi quipehualtía cualantli huan quichihua miyac tlamantli tlajtlacoli. 
Se masehuali catli mohueyimati quipantis catli quiechcapantlalis, pero catli moechcapanohuaj quipantise tlatlepanitacayotl. 
Se masehuali catli quipalehuía se tlachtejquetl para tlachtequis amo quitlepanita inemilis. Quicaqui quema nopa juez quiilhuía para ma tlatestigojquetza para quiijtos catli xitlahuac huan amo quimati sintla quisas más cuali para quiijtos catli xitlahuac o noja quitlatis catli quichijqui. 
Catli más quinimacasij masehualme huan amo TOTECO mocalaquíaj ipan se tlaquetztli, pero catli motemachíaj ipan TOTECO itztose ica temachili. 
Miyac masehualme quitlajtlaníaj nopa tlanahuatijquetl ma quinchihuili se favor para ma quinpalehuis quema monequi quintlajtolsencahuas, pero más cuali quitemose TOTECO pampa yaya nochipa tetlajtolsencahua xitlahuac. 
Masehualme catli monejnemiltíaj xitlahuac amo quinequij quinitase amo cuajcualme. Huan amo cuajcualme nojquiya quincualancaitaj masehualme catli nemij ipan se ojtli catli xitlahuac.
