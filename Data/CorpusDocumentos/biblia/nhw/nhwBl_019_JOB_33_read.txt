JOB.
33.
“Job, na nimitztlajtlanía xijtlacaquili catli nimitzilhuis. 
Na nipejtoc nicamanalti, huajca techcahuili ma nijsenhuiquilis. 
Huan niquijtos nochi catli senquistoc xitlahuac huan catli melahuac. 
Itonaltzi Toteco nechchijchijqui. Quena, tlaijiyoquixti Toteco Catli Quipiya Nochi Chicahualistli huan nechmaca nemilistli ica iijiyo. 
Ximocualtlali huan technanquili nimantzi sintla tihueli. 
“Na nojquiya niitztoc san se quen ta iixpa Toteco. Na nojquiya nitlachijchihuali ica soquitl quen ta. 
Huajca amo techimacasi na pampa na amo nieli se masehuali catli quipiya hueyi tequiticayotl para ica nimitzmajmatis o nimitzchihuas xihuihuipica. 
“Na nimitztlacaquilijtoya quema tiquijto, 
para titlapajpactic huan amo tleno amo cuali tijchijtoc. Tiquijtojtoc tixitlahuac huan amo tijpiya tlajtlacoli. 
Tiquijtohua para Toteco san quitemohua quenicatza huelis quipantis se tlajtlacoli para mitzitas quen tiicualancaitaca. 
Nojquiya tiquijtojtoc para Toteco quicalaquijtoc moicxihua ipan cuahuitl para quintetzopas, huan mitztlachilía sesen huelta quema timolinía. 
“Xiquita, ya ni catli na nimitzilhuía. Melahuac titlajtlacolchijtoc pampa amo xitlahuac ticamanaltitoc tlen Toteco. Toteco Dios itztoc más hueyi que se masehuali. 
Huajca ¿para tlen tijtelhuía ica catli amo cuali pampa yaya amo mitzyolmelahua para tlen mitztlatzacuiltía? 
“Toteco Dios technojnotza tlatoctzitzi, pero tojuanti amo tijmachilíaj. 
Quemantic technojnotza ipan se temictli, o ipan se tlanextili ica tlayohua quema masehualme tlahuel cochtoque ipan inintlapech. 
Toteco tecamanalhuía para quinconsejomacas huan para quinmajmatis ica catli panos teipa. 
Quinnotza masehualme para quinyolcuepas tlen inintlajtlacolhua huan para ma amo mohueyimatica. 
Nojquiya quinmanahuía ininnemilis para amo huetzise imaco catli amo cuali huan para amo yase mictla. 
“Pero nojquiya quemantic Toteco quixitlahua se masehuali ica cocolistli huan tlacuajcualocayotl ipan nochi itlacayo, masque amo quipostequi niyon se iomiyo. 
Huan nopa cocoxquetl ayacmo tleno quiixtocas para quicuas masque tlahuel ajhuiyac nopa tlacualistli pampa ayacmo mayanas. 
Quej nopa nopa masehuali mocahuas tlahuel cuahuactoc huan san omitl, 
huan monechcahuis nopa tonal para miquis huan quitlalpachose. 
“Pero sintla itztosquía se tlayolmelajquetl tlen ilhuicac catli motlatlajtisquía ica Toteco huan quinextisquía para nopa tlacatl itztoc xitlahuac, 
huajca Toteco quitlasojtlasquía huan quiilhuisquía itequipanojca: ‘Xijmajcahuaca ma yohui nopa masehuali. Amo xijchihuaca ma miqui pampa nijpantijtoc seyoc catli tlaxtlahuas por inemilis.’ 
Huan huajca itlacayo nopa masehuali mochicahuas quen se conetl itlacayo. Quipiyas chicahualistli quen se telpocatl. 
Huan quema motlatlajtis, Toteco quitlacaquilis huan quiicnelis. Huan yaya moixtlachilis ica Toteco huan huicas ica miyac paquilistli. Huan Toteco sempa quiijtos para xitlahuac huan quitiochihuas miyac. 
Huan nopa masehuali quinilhuis ihuampoyohua: ‘Nitlajtlacolchijqui huan nitlaixpanotoc, pero Toteco amo nechmacac catli quinamiqui. 
Toteco nechmanahui tlen miquilistli huajca noja niitztos ipan ni tlaltipactli ipan tlahuili.’ 
“Quena, Toteco quichihua ya ni miyacpa para quinojnotzas se masehuali. 
Yaya quimanahuía ialma tlen miquilistli para huelis itztos ininhuaya catli yoltoque nica campa tlahuili. 
Ximomaca cuenta catli niquijtohua, Job. Techtlacaquili huan noja techcahuili ma niquijto se ome camanali. 
Pero sintla ta tijnequi tiquijtos se camanali, huajca xiquijto. Huan na nijtlacaquilis huan nimitzilhuis sintla xitlahuac o amo. 
Huan sintla amo tleno tijnequi tiquijtos, huajca techtlacaquili. Amo ximolini huan na nimitzmachtis para titlalnamiquis.”
