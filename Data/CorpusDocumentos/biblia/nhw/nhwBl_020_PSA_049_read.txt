SALMOS.
49.
Cuali xijtlacaquilica ya ni, amojuanti catli campa hueli anitztoque ipan ni tlaltipactli. 
Nocamanal elis para nochime. Elis para catli tominpixtoque huan para catli teicneltzitzi. Elis para catli quipiyaj tequiticayotl huan para catli amo quipiyaj tequiticayotl. Huajca xijtlacaquilica. 
Nicamanaltis ica tlalnamiquilistli. Nimechpohuilis nochi catli nimonejnehuilijtoc ipan noyolo. 
Achtihui nimechilhuis se tlatzintoquili catli totlalnamijca huejcapan tatahua quiijtojque, huan nitlatzotzonas ica se arpa huan nihuicas para nimechmatiltis tlaque quinequi quiijtos: 
¿Para tlen monequi nimomajmatis quema hualas tlaohuijcayotl, huan quema nopa tlajtlacolchijca masehualme catli nechcualancaitaj hualase nechyahualoquij? 
Inijuanti motemachiyaj ipan inintomi, huan mohueyimatij pampa quipiyaj miyac tlamantli catli pajpatiyo. 
Pero niyon se tlen inijuanti amo huelis momaquixtis tlen itlajtlacolhua, niyon amo huelis quimaquixtise se iniicni. Amo aqui huelis quitlaxtlahuis Toteco Dios para ma quimaquixtis. 
Pampa tonemilis eltoc se tlamantli catli miyac ipati. Amo onca niyon se tomi ipan nochi tlaltipactli catli ica huelis moixtlahuas. 
¡Amo! Amo aqui huelis quicohuas inemilis para itztos para nochipa huan para amo quema miquis. 
Nochi masehualme miquise. Catli tlalnamiquij huan catli mohueyimatij, huan nojquiya nochi catli huihuitique, san se ininpantis. Mictehuaj huan quicajtehuaj nochi catli pajpatiyo quipiyayayaj. Huan sequinoc moaxcatise. 
Inijuanti moilhuíaj itztose para nochipa, yeca quicualtlalíaj ininchajchaj huan quitlalíaj inintoca ipan inintlal, quen elisquía iniaxca para nochipa. 
Pero se masehuali amo huejcahuas masque nelía tominpixtoc, pampa nojquiya miqui san se quen se cuatitlan tlapiyali. 
Nochi masehualme catli motemachijtoque san ipan ininseltzitzi, quipantise miquilistli. Huan nojquiya miquise nopa masehualme catli quintoquilíaj huan quihueyitlalíaj inincamanal. 
Nopa masehualme miquise quen miquij borregojme. Quena, miquilistli quinchiya nochime. Pero se tonal nopa masehualme catli itztoque xitlahuaque iixpa Toteco Dios quinnahuatise masehualme catli amo cuajcualme. Quej nopa elis quema ya palantose inintlacayohua nopa masehualme ostoijtic. Huajca quena, ipan nopa tonali nopa tominpiyani quicajtejtose huejca ininchajchaj catli yejyectzitzi. 
Pero Toteco Dios nechmaquixtis tlen ichicahualis miquilistli huan nechhuicas ihuaya. 
Amo ximotequipacho sintla se masehuali moricojchihua, huan mojmostla más quiyecchijchihua ichaj. 
Pampa quema miquis amo tleno huelis quihuicas. ¡Amo! Quema quitemohuise itlacayo ostoijtic, niyon quentzi catli yejyectzi quipixqui amo huelis yas ihuaya. 
Masque se masehuali itztos ica paquilistli quema noja yoltoc, huan sequinoc quitlepanitase pampa moricojchijtoc, 
ajsis se tonal quema miquis quen ihuejcapan tatahua, huan ayacmo quema quiitas tlahuili. 
Se masehuali catli moricojchijtoc, pero amo quielnamiqui para amo huejcahuas inemilis ipan ni tlaltipactli, mictehuas san se quen se cuatitlan tlapiyali.
