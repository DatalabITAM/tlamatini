PROVERBIOS.
10.
Ya ni nopa tlalnamijca camanali catli Salomón quiijcuilo. Se telpocatl catli tlalnamiqui quipaquilismaca itata huan inana, pero se catli quej hueli nemi, quinmaca miyac tequipacholi. 
Tlamantli catli tlaichtectli amo temaca paquilistli, niyon amo huejcahua. Pero nochi masehualme catli nemij xitlahuac iixpa Toteco momanahuise tlen se miquilistli catli fiero. 
TOTECO amo quicahuas para se masehuali catli nemi xitlahuac ma mayancamiqui, niyon amo quicahuilis se masehuali catli amo quitlepanita TOTECO ma ixhuis. Nochipa quinequis más. 
Se catli tlatzihui nimantzi mochihuas se catli teicneltzi, pero se catli tequiti chicahuac, moricojchihuas. 
Tlalnamiqui nopa tlacatl catli quisentilía itlacualis ipan nopa tonalme quema noja tlatotoniya. Pero catli cochi ipan nopa hora para pixcas, mopinahuas quema amo quiaxilis itlacualis. 
TOTECO quimacas miyac tlatiochihualistli se tlacatl catli nemi xitlahuac iixpa, pero amo cuajcualme quipantíaj miyac tlamantli tlahuilancayotl por catli fiero quichihuaj. 
Quema miquis se tlacatl catli xitlahuac, nochi quielnamiquij ica cuali. Pero quema miquise amo cuajcualme, catli quinelnamiquij quinitaj quen se tlamantli catli pojyajtoc. 
Se catli tlalnamiqui quiselía tlanahuatili ica paquilistli, pero se catli huihuitic camanalti miyac huan nima onhuetzi ica ixayac ipan tequipacholi. 
Se catli monejnemiltía xitlahuac quipiya temachili ipan inemilis. Pero se catli nemi ipan ojtli catli huijcoltic motepotlamis huan tlahuel mococos. 
Masehualme catli amo tleno quichihuaj quema se tlacatl quiixpano ne seyoc, tlahuel temacaj tequipacholi. Pero catli quinajhuaj chicahuac catli fiero quichihuaj quiseseltíaj cualantli. 
Quema camanalti se tlacatl catli itztoc xitlahuac eltoc quen se ameli catli temaca se nemilistli catli cuali. Pero ipan inincamac amo cuajcualme quisa catli quipehualtía cualantli. 
Se catli quincualancaita sequinoc quipehualtía cualantli. Pero catli teicnelía quintlapojpolhuía catli quiixpanoj. 
Quema se masehuali catli tlalnamiqui camanalti, nochi catli quicaquij momacaj cuenta para quimachilía miyac tlamantli. Pero quema se catli huihuitic camanalti, nochi momacaj cuenta para monequi aqui quihuitequis ica se cuahuitl. 
Masehualme catli tlalnamiquini nochipa quinequij quimatise más tlamantli. Pero se catli huihuitic camanalti quej hueli huan san motlanilía tequipacholi. 
Se catli tominpiya huelis momanahuis ica itomi para itztos ica temachili. Pero se catli teicneltzi motequipachohua pampa amo tleno quipiya para ica mopalehuisquía. 
Nochi catli quitlani se masehuali catli itztoc xitlahuac quipalehuía para quipiyas catli cuali ipan inemilis. Pero amo cuajcualme san quinenpolohuaj inintomi ipan tlajtlacoli. 
Se masehuali catli quitlacaquilía quema seyoc quixitlahua nemi ipan nopa ojtli catli yohui campa onca nemilistli. Pero se catli amo quitlacaquilía catli quinequi quitlalnamictis, mocuapolos. 
Se catli tecualancaita huan mochihua quen teicnelía, yaya se istlacatiquetl. Huan se catli tetlaijilhuía nelía huihuitic. 
Se catli tlahuel camanalti amo polihui para quiijtos se camanali catli ica tlaixpanos. Pero se catli tlalnamiqui quipiya miyac cuidado ica catli quiijtohua. 
Quema se tlacatl catli itztoc xitlahuac iixpa TOTECO camanalti, monequi tijtlacaquilis pampa icamanal ipati quen plata catli más cuali. Pero inincamanal amo cuajcualme amo tleno ipati. 
Masehualme catli nemij xitlahuac iixpa TOTECO hueli temacaj cuali consejos. Pero catli huihuitique miquise pampa amo quipiyaj tlalnamiquilistli para monejnemiltise cuali. 
Quema TOTECO quitiochihuas se masehuali, yaya moricojchihuas ica catli nelía ipati. Huan TOTECO quimaca miyac paquilistli. 
Se tlacatl catli huihuitic quipaquilismaca para quichihua catli fiero. Pero se catli tlalnamiqui quipaquilismaca para quipiyas más tlalnamiquilistli. 
Nopa catli tlahuel quiimacasi se masehuali catli amo cuali, ya nopa catli quipantis. Pero masehualme catli itztoque xitlahuaque iixpa TOTECO, yaya quinmacas nopa tlamantli catli quinequij. 
Quena, quen se hueyi ajcomalacatl san hualpanotiquisa, quej nopa nojquiya itztoque masehualme catli amo quitlepanitaj TOTECO pampa amo huejcahuaj para itztose. Pero masehualme catli itztoque xitlahuac iixpa TOTECO huejcahuase pampa moquetztoque ipan se tlatzinpehualtili catli temachtli. 
Se tetequipanojquetl catli tlatzihui tlahuel quitequipachohua iteco quen poctli catli calaqui ipan iixteyol, o quen vinagre catli quiajsi itlancoch hasta cuajcualo. 
Quema se masehuali quiimacasi huan quitlepanita TOTECO, TOTECO quinchihuas para ma itzto miyac xihuitl. Pero TOTECO quincotonilis ininxihui catli amo cuajcualme. 
Catli itztoque xitlahuaque iixpa TOTECO quichiyaj tlamantli catli temaca paquilistli. Pero masehualme catli amo quitlepanitaj TOTECO quichiyaj tlamantli catli san tlapic. 
TOTECO quinmocuitlahuía catli nemij xitlahuaque iixpa, pero quintzontlamiltía masehualme catli quichihuaj catli amo cuali. 
TOTECO quintiochihuas catli itztoque xitlahuaque iixpa huan amo quema tlatzintlatemose. Pero quinquixtis masehualme catli amo quitlepanitaj ipan ni tlali, pampa amo quincahuilis ma huejcahuaca para itztose. 
Quema camanalti se masehuali catli itztoc xitlahuac iixpa TOTECO, tetlalnamictía. Pero se tecajcayajquetl, TOTECO quitzontlamiltis para ayacmo huelis camanaltis. 
Quema camanalti se cuali tlacatl, icamanal tepalehuía. Pero amo cuajcualme quiijtohuaj camanali catli tlacajcayahuaj.
