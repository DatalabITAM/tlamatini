JOB.
9.
Job quiilhui Bildad: 
“Na nijmati para melahuac Toteco amo quihuejcamajcahua catli xitlahuac. Pero ¿quenicatza huelis se masehuali mochihuas xitlahuac iixpa Toteco Dios? 
Sintla se masehuali quinequisquía monajnanquilis ihuaya Toteco, amo hueli quinanquilis se huelta ipan se mil itlatzintoquilijcahua. 
Pampa Toteco Dios tlahuel tlalnamiqui huan tlahuel quipiya chicahualistli. ¿Ajqueya mohuilanqui ihuaya huan tlatlanqui? 
“Ica ihueyi cualancayo Toteco quinijcuinía tepeme. Quinixcuepa ipan se tlatoctzi quema amo aqui momaca cuenta. 
Quicocoxonía ni tlaltipactli hasta campa itzinpehualtil. Quichihua ma tlahuel mojmoliníaj itlaquetzalhua. 
Sintla Toteco quimaca tlanahuatili, tonati amo cahuanis, huan sitlalime amo tlahuise. 
Toteco iseltzi quipatlajqui ilhuicactli. Yaya nejnemi ipan hueyi atl para quitamachihuas ihuexca. 
Toteco quinchijchijqui nochi sitlalime hasta inijuanti catli nejnemij motzquitoque huan inintoca Osa Mayor, Orión huan nopa Pléyades. Nojquiya quinchijchijqui nopa miyaqui sitlalime catli motzquitoque san sejco ica tlani ipan ilhuicactli. 
Toteco quichihua huejhueyi tlamantli catli amo aqui huelis quimachilis. Quichihua tlahuel miyac tiochicahual nextili hasta amo aqui huelis quipohuas. 
“Pero quema yaya pano noixpa, amo niquita. Quema yaya nentinemi nonechca, amo nijmachilía. 
Quema Toteco quititlani miquilistli para quiitzquis se masehuali, ¿ajqueya hueli quitzacuilis para ma amo quichihua? O ¿ajqueya mosemacas para quitlatzintoquilis: ‘¿Tlaque tijchihua?’ 
Toteco Dios amo quitzacuilía icualancayo. Iixpa ya moechcapanose nochi catli mohueyimatij. 
“Huajca ¿ajqueya na para nimonajnanquilis ihuaya Toteco Catli Quipiya Nochi Chicahualistli? ¿Ica tlaque camanali huelis nimomanahuis iixpa? 
Masque amo tleno nijchijtoc, amo hueli nimomanahuis ica miyac camanali. San nijtlajtlanisquía ma nechtlasojtla huan ma quipiya hueyi iyolo ica na, pampa yaya Juez. 
Masque nijnojnotzasquía huan yaya nechnanquilisquía, amo nijneltoca para nelía nechtlacaquilisquía. 
Yaya quimiyaquilisquía notlaijiyohuilis huan nechtzontlamiltisquía, masque amo tleno nijchijtoc. 
Yaya amo nechcahuilía ma nitlaijiyotilana. San onpano nechtlaijiyohuiltía. 
Sintla ticamanalti tlen tetili, Toteco Dios catli más quipiya fuerza. Sintla ticamanaltisquía tlen ajqueya más xitlahuac, yaya tlatlanis. Amo aqui hueli quitelhuis ica niyon se tlamantli. 
Masque na niquijtosquía amo nijhuica tlajtlacoli, ica nocamac nimotelhuisquía. Huan masque niquijtosquía para amo aqui huelis nechtelhuis, Toteco Dios nechilhuisquía para senquistoc nitlajtlacolchijquetl. 
“Masque amo tleno nijchijtoc, niquita para amo tleno ipati. ¡Niquijiya nonemilis! 
Toteco quinita catli xitlahuaque huan catli tlajtlacolchihuaj quen san se. Yeca niquijtohua para san se quintzontlamiltía catli cuali huan catli amo cuali. 
Quema huala se fiero cocolistli ipan masehualme catli amo tleno quichijtoque, yaya san huetzca. 
Quema se tlali huetzi ininmaco tlacame catli mosisiníaj, Toteco Dios quinixtzacua nopa tetlajtolsencahuani huan quincahua ma quichihuaca catli amo xitlahuac. Sintla amo Toteco Dios catli quincahua ma quichihuaca catli amo xitlahuac, huajca, ¿ajqueya quichihua? 
“Nonemilis panoti chicahuac quen se motlalojquetl. Quena, nimantzi pano huan amo niquita se tlatoctzi tlen paquilistli. 
Quen se pilcuaacaltzi yohui chicahuac ipan atl huan se cuajtli chicahuac patlani para quiitzquis se piyo, quej nopa nojquiya nimantzi panoti nonemilis. 
Masque nijchihuilía fuerza para niquelcahuas notlaijiyohuilis huan nijcahuasquía notequipachol huan nijchihuasquía para niyolpactos, amo nechpalehuisquía. 
“Ta Toteco noja más tlaijiyohuilistli techtitlanisquía. Pampa nijmati ta, tiToteco Dios, techita para amo nixitlahuac moixpa. 
Masque nijchihua catli cuali o catli amo cuali, ta temachtli techtlatzacuiltis. Huajca ¿para tlen nijchihuas campica? 
Masque nimaltisquía ica atl huan nimomajtequisquía ica nopa xapo catli más tlapajpaca, 
ta techcalaquisquía ipan se soqui ostotl huan techchihuasquía nelía nisoquiyo hasta noyoyo nojquiya nechijiyasquía. 
“Toteco amo ya se tlacatl quen na para huelis nimonajnanquilis ihuaya. Amo hueli nijhuicas iixpa se juez para nijtelhuis. 
Nipaquisquía sintla itztoya se catli hueli motlajcoquetzas tlatlajco na huan Toteco Dios para techyoltlalisquía, pero amo aqui. 
Sintla itztosquía se catli motlajcoquetza, quichihuasquía para Toteco Dios ma ayacmo nechtlaijiyolhuiltisquía, huan ayacmo niquimacasisquía itlatzacuiltilis. 
Huajca nihuelisquía nijcamanalhuisquía Toteco huan amo niquimacasisquía. Pero amo hueli nijchihuas san ica na nochicahualis.
