NÚMEROS.
33.
Nica eltoc inintoca nochi nopa lugares campa nopa israelitame panotiyajque quema quisque ipan tlali Egipto ica nochi nopa huejhueyi pamitl tlen ininmasehualhua, huan quinyacantiyohuiyayaj Moisés ihuaya Aarón. 
Moisés quiijcuilojtiyajqui nochi inintoca nopa lugares campa quiquetztiyajque ininyoyoncalhua. 
Nopa israelitame quistejque nopa altepetl Ramesés ipan tlali Egipto ipan 14 itequi nopa achtihui metztli ipan nopa israelita calendario 33:3 Ni tonal quemantic huetzi ipan marzo huan quemantic ipan ipejya abril ipan tocalendario tlen ama.. Quinextijque nopa ilhuitl Pascua huan ipan ijnaloc tlen ne seyoc tonali quistejque ica hueyi chicahualistli iniixtla nopa egiptome. 
Huan quema quistejque, nopa egiptome noja quintlaltocayayaj ininconehua catli achtihuime pampa TOTECO quichijtoya ma miquica. Huan TOTECO nojquiya quintlamitepejqui nochi inintlaixcopincayohua. 
Huan nopa israelitame quistejque altepetl Ramesés huan quiquetzque ininyoyoncalhua ipan Sucot. 
Quistejque Sucot huan mochihuatoj ipan Etam campa pehua nopa huactoc tlali. 
Quistejque Etam huan mocuepque para quimacase se huelta ica Pi Hahirot catli mocahua ica campa hualquisa tonati tlen Baal Zefón, huan nopona mochijque nechca Migdol. 
Huan quema quistejque Pi Hahirot, quiixcotonque nopa hueyi atl huan ajsitoj campa nopa huactoc tlali. Huan nejnentiyajque eyi tonali ipan nopa huactoc tlali tlen Etam huan mochijque campa Mara. 
Huan quistejque tlen Mara huan ajsitoj ipan nopa huactoc tlali catli itoca Elim campa oncayaya 12 amelme huan 70 apachi cuame, huan mochijque nopona. 
Teipa quistejque Elim huan mochihuatoj nechca nopa hueyi atl catli itoca Chichiltic. 
Huan quistejque campa nopa Hueyi Atl Chichiltic huan mochihuatoj ipan nopa huactoc tlali catli itoca Sin. 
Huan quistejque campa nopa huactoc tlali Sin huan mochihuatoj ipan Dofca. 
Huan quistejque Dofca huan mochihuatoj ipan Alús. 
Teipa quistejque Alús huan mochihuatoj ipan huactoc tlali tlen Refidim campa amo oncayaya atl para quiijise. 
Huan quistejque Refidim huan mochihuatoj ipan nopa huactoc tlali Sinaí. 
Teipa quistejque nopa huactoc tlali Sinaí huan mochihuatoj ipan Kibrot Hataava. 
Huan quistejque Kibrot Hataava huan mochihuatoj ipan Hazerot. 
Huan quistejque Hazerot huan mochihuatoj ipan Ritma. 
Teipa quistejque Ritma huan mochihuatoj ipan Rimón Peres. 
Huan quistejque Rimón Peres huan mochihuatoj ipan Libna. 
Huan quistejque Libna huan mochihuatoj ipan Rissa. 
Huan quistejque Rissa huan mochihuatoj ipan Ceelata. 
Teipa quistejque Ceelata huan mochihuatoj ipan nopa tepetl Sefer. 
Huan quistejque campa nopa tepetl Sefer huan mochihuatoj ipan Harada. 
Huan quistejque Harada huan mochihuatoj ipan Macelot. 
Teipa quistejque Macelot huan mochihuatoj ipan Tahat. 
Huan quistejque Tahat huan mochihuatoj ipan Tara. 
Teipa quistejque Tara huan mochihuatoj ipan Mitca. 
Huan quistejque Mitca huan mochihuatoj ipan Hasmona. 
Huan quistejque Hasmona huan mochihuatoj ipan Moserot. 
Huan quistejque Moserot huan mochihuatoj ipan Bene Jaacán. 
Huan quistejque Bene Jaacán huan mochihuatoj ipan Hor ipan tlali Gidgad. 
Huan quistejque Hor ipan tlali Gidgad huan mochihuatoj ipan Jotbata. 
Huan quistejque Jotbata huan mochihuatoj ipan Abrona. 
Huan quistejque Abrona huan mochihuatoj ipan Ezión Geber. 
Huan quistejque Ezión Geber huan mochihuatoj ipan huactoc tlali tlen Zin campa Cades. 
Huan quistejque Cades huan mochihuatoj ipan nopa tepetl Hor campa monepantía nopa tlali Edom. 
Huan quen TOTECO quinahuati ma quichihua, tlejcoc nopa totajtzi Aarón ipan ne tepetl Hor huan nopona mijqui ipan nopa achtihui tonal ipan nopa macuilpa metztli ipan nopa israelita calendario 33:38 Ni tonal eltoc ipan julio ipan tocalendario tlen ama., quema ya quipiyayaya 40 xihuitl para quistoyaj ipan tlali Egipto. 
Huan quema Aarón mijqui ipan nopa tepetl Hor, quipiyayaya 123 xihuitl. 
Huan Tlanahuatijquetl Arad catli itztoya ipan nopa huactoc tlali Neguev ipan tlali Canaán quicajqui para ajsitihualayayaj nopa israelitame. 
Huan nopa israelitame quistejque tepetl Hor huan mochihuatoj ipan Zalmona. 
Huan quistejque Zalmona huan mochihuatoj ipan Punón. 
Huan quistejque Punón huan mochihuatoj ipan Obot. 
Huan quistejque Obot huan mochihuatoj ipan Ije Abarim campa monepantía ica tlali Moab. 
Huan quistejque Ije Abarim huan mochihuatoj ipan Dibón Gad. 
Huan quistejque Dibón Gad huan mochihuatoj ipan Almón Diblataim. 
Huan quistejque Almón Diblataim huan mochihuatoj ipan nopa tepeme tlen Abarim catli mocahua más nechca campa hualquisa tonati tlen Nebo. 
Huan quistejque nopa tepeme tlen Abarim huan mochihuatoj ipan nopa tlamayamitl ipan tlali Moab nechca nopa hueyatl Jordán, huan iixmelac altepetl Jericó catli mocajqui ipan seyoc nali. 
Huan quiquetzque ininyoyoncalhua nechca hueyatl Jordán ipan nopa tlamayamitl tlen tlali Moab, huan ininchajchaj pejqui campa Bet Jesimot huan ajsic hasta Abel Sitim. 
Ipan nopa tlamayamitl ipan tlali Moab nechca hueyatl Jordán, iixmelac altepetl Jericó, TOTECO quicamanalhui Moisés huan quiilhui: 
“Xiquinilhui nopa israelitame ya ni: Quema anquiixcotonase ni hueyatl Jordán huan ancalaquise ipan tlali Canaán, 
xiquinmictica nochi catli itztoque ipan nopa tlali. Xiquinsosolhuilica nochi inindioses masque quichijtoque tlen tetl, o quitejtejtzontoque tlen tepostli. Nochi inintlaixcopincayohua huan inintiotzitzi xiquinsosolhuilica huan nojquiya xijsosoloca nochi tlaixpamitl campa tlacajcahuaj ipan tepeme. 
Ximocuilica nopa tlali huan nopona ximochantica para anitztose, pampa na nimechmactilía para elis amoaxca. 
Pero monequi ximomajmacaca nopa tlali ica sorteo quen imiyaca imasehualhua ipan sesen hueyi familia. Nopa huejhueyi familias ica más miyac tlacame quichihuase se sorteo para nopa tlalme catli achi hueyi, huan nopa huejhueyi familias catli amo quipiya más miyaqui tlacame quichihuase sorteo para nopa tlalme catli amo más hueyi. Huan ijquiyampa, nopa sorteo quinextis para elis iaxca se familia, ya nopa elis iaxca. 
“Pero sintla amo anquintlamimictise catli ama itztoque ipan nopa tlali, catli anquincahuilise ma mocahuaca tlahuel anmechcuesolmacase quen se cuatejtli catli calaqui ipan amoixteyol o quen se huitztli catli calaqui ipan amotlacayo. Huan miyac antlaijiyohuise ipan nopa tlali catli nimechmaca. 
Huan na nimechtlatzacuiltis amojuanti quen nimoilhuiyaya niquintlatzacuiltis inijuanti.”
