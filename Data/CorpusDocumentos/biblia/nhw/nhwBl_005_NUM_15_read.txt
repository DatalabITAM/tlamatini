NÚMEROS.
15.
Huan TOTECO quicamanalhui Moisés huan quiilhui: 
“Xiquincamanalhui nopa israelitame huan xiquinilhui quema calactiyase ipan nopa cuali tlali catli niquinmacas para ipan itztotij, 
huan quinequise nechpaquilismacase ica se tlacajcahualistli tlatlatili, o se tlacajcahualistli catli nechmacase ipan tlitl, quej ni monequi quichihuase. Masque nechmacase nopa tlacajcahualistli pampa quitlamichijtoque se tlenijqui catli nechtencahuilijtoque quichihuase, o pampa huelis san ica ininpaquilis quinequij nechmacase se tlenijqui, o pampa ajsis se ilhuitl quema monequi se tlacajcahualistli, o quinequij nechyolpactise ica catli mijyotis ajhuiyac noixpa, huajca monequi nechmacase se borrego, se chivo, o se becerro. 
Ihuaya sesen tlacajcahualistli monequi nojquiya nechmacase nopa tlacajcahualistli tlen harina catli ihuaya yohui. Sintla se pilborregojtzi nechmacase, monequi ihuaya nechmacase ome kilo catli cuali harina tlamaneloli ica se litro aceite. 
Huan sintla nechmacase se pilborregojtzi quen se tlacajcahualistli tlatoyahuali, nechmacase se litro xocomeca atl quen se tlacajcahualistli tlatoyahuali huan mijyotis cuali noixpa. 
Huan sintla nechmacase se oquich borrego, ihuaya nechmacase nojquiya nahui kilos catli cuali harina tlamaneloli ica se litro huan se achi tlen aceite. 
Huan nojquiya ihuaya nechmacase se litro huan se achi tlen xocomeca atl quen se tlacajcahualistli tlatoyahuali noixpa. 
Huan sintla nechtencahuilise se becerro para se tlacajcahualistli tlatlatili, o pampa quitlamiltijtoque catli nechtencahuilijtoque quichihuase, o sintla nechmacase pampa quinequij quipiyase tlayoltlalili nohuaya, 
monequi ihuaya nopa becerro nechmacase se tlacajcahualistli tlen chicuase kilos catli cuali harina tlamaneloli ica ome litros tlen aceite. 
Huan ihuaya nojquiya nechmacase ome litros tlen xocomeca atl quen se tlacajcahualistli tlatoyahuali noixpa. Huan nochi san sejco quitlatise huan mijyotis ajhuiyac noixpa huan nechpactis. 
Huajca quej ni sese israelita monequi quisencahuas sesen becerro, sese oquich borrego, sesen pilborregojtzi, o sesen chivo catli nechmacase. 
Nochi israelitame quichihuase nochi ya ni quema nechmacase se tlacajcahualistli catli quitlatíaj para ajhuiyac niquijnecuis. 
“Huan san se quichihuas se catli seyoc tlali ejquetl catli itztos amohuaya, huan quinequis nojquiya nechtencahuilis se tlacajcahualistli catli quitlatis. Nochi san se tlanahuatili quitoquilise para quenicatza cuali quisencahuase inintlacajcahualis para na. 
Israelitame huan seyoc tlali ehuani catli mocahuaj ininhuaya san se tlanahuatili quitoquilise para nochipa. 
Quej nopa anisraelitame huan seyoc tlali ehuani san se anitztose noixpa.” 
Huan TOTECO quicamanalhui Moisés huan quiilhui: 
“Xiquinilhui nochi israelitame quema calactose ipan nopa cuali tlali campa na niquinhuicas, 
huan pehuase quicuase nopa trigo catli temaca nopa tlali, monequi quiiyocatlalise se achi para nechtencahuilise. 
Quema achtihui quisentilise trigo sesen xihuitl huan quichihuase nopa achtihui tixtli, ma nechsencahuilica se pantzi huan quihuisose noixpa quen se tlacajcahualistli tlen pantzi tlen nopa trigo pixquistli. 
Huan quitencahuase ni tlacajcahualistli tlen nopa achtihui trigo ipan sesen xihuitl hasta iniixhuihua quinpiyase iniixhuihua huan para nochipa. 
“Huan masque nelía quinequise quichihuase nochi senquistoc cuali, sintla inijuanti o iniixhuihua teipa ehuani, amo quichihuase nochi ni tlanahuatilme catli nimechmacatoc ica Moisés hasta quema nipejqui nijmaca tlanahuatilme, huajca ya ni catli monequi quichihuase. 
Quema san sejco inijuanti momacase cuenta para amo nochi tlamantli catli quichihuayayaj eltoya senquistoc cuali noixpa, monequi nechtencahuilise se becerro quen se tlacajcahualistli tlatlatili catli tlahuel ajhuiyac mijyotis para na. Huan ihuaya nechmacase itlacajcahualis tlen harina huan tlen xocomeca atl quen quiijtohua nopa tlanahuatili catli nimechilhuijtoc. Huan nojquiya nechtencahuilise se chivo quen se tlacajcahualistli por nopa tlajtlacoli catli quichijque. 
Huajca nopa totajtzi nechmacas nopa tlacajcahualistli para nechixtzacuilis inintlajtlacolhua nochi israelitame huan niquintlapojpolhuis, pampa ya nechtencahuili nopa tlacajcahualistli por nopa tlajtlacoli catli quichijque masque amo quimatiyayaj. 
Huan na niquintlapojpolhuis nochi israelitame huan nochi seyoc tlali ehuani catli itztose ininhuaya por nopa tlajtlacoli catli nochi inijuanti sentic quichijque masque amo quimatiyayaj. 
“Sintla tlajtlacolchihuas se masehuali san pampa amo quimati, quema momacas cuenta catli quichijqui, quitencahuas se sihua chivo catli san quipiya se xihuitl quen se tlacajcahualistli por tlajtlacoli. 
Huan nopa totajtzi nechmacas nopa tlacajcahualistli para nechixtzacuilis itlajtlacol noixpa pampa amo quimatiyaya sintla quichihuayaya se tlajtlacoli, huan na nijtlapojpolhuis. 
Huan ni tlanahuatili monequi san se quitoquilise israelitame huan catli seyoc tlali ehuani catli itztose amohuaya quema quichihuaj tlajtlacoli pero amo quimatiyayaj. 
“Pero sintla tlajtlacolchihua se tlacatl ica ipaquilis, masque se israelita o amo, yaya tlahuel nechhuejcamajcahua na, niamoTECO, huan monequi miquis para ayacmo itztos amohuaya. 
Anquimictise pampa amo quinejqui quicaquis nocamanal huan amo quineltocac notlanahuatil, huan nochi itlajtlacol huetzis ipan ya.” 
Quema nopa israelitame itztoyaj ipan nopa huactoc tlali, quipantijque se tlacatl quisentiliyaya cuahuitl ipan se tonal para mosiyajquetzase. 
Huan inijuanti catli quipantijque quihuicaque iixpa Moisés ihuaya Aarón huan iniixpa nochi israelita tlayacanani, 
huan quitlalijque ipan tlatzactli pampa ayamo quimatiyayaj tlaque ipaquilis TOTECO para quichihuilise. 
Pero teipa TOTECO quiilhui Moisés: “Nopa tlacatl monequi miquis. Nochi anisraelitame monequi anquiquixtise huejca tlen amoaltepe campa anquinquetztoque amoyoyoncalhua huan anquitepachose hasta miquis.” 
Huajca nopa israelitame quiquixtijque nopa tlacatl tlen inialtepe huan quihuicaque huejca. Huan quitzontepachojque hasta quimictijque senquistoc quen TOTECO quinahuatijtoya Moisés. 
Huan TOTECO quicamanalhui Moisés, huan quiilhui: 
“Xiquincamanalhui nochi israelitame, huan xiquinilhui ma mochihuilica inintenhuaxo para quiilpise ipan itencuelpachol nochi ininyoyo. Monequi quitlalise itenhuaxo ica icpatl catli azultic. 
Huan quema anquitlachilise itenhuaxo amoyoyo, anquielnamiquise itlanahuatil TOTECO huan anquitoquilise. Huan amo antechcajtehuase quen se tlacanenca sihuatl quicahua ihuehue. Niyon amo anquichihuase san catli anquinequij anquichihuase, o catli anmoilhuíaj para cuali. 
Quej nopa anquielnamiquise nochi notlanahuatil huan anquineltocase, huan aneltose antlatzejtzeloltique san para na, niamoTeco Dios. 
Na niDIOS amoTeco, huan na catli nimechquixti ipan tlali Egipto para antechiximatise para niamoTECO. Quena, na niamoTECO Dios.”
