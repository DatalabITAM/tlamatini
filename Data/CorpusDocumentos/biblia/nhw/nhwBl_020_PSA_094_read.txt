SALMOS.
94.
TOTECO, ximonexti. Ta tiDios catli tijpiya motequi para tiquincuepilis catli amo cuali techchihuilijtoque tocualancaitacahua. 
Ximotlanana huan xijtlatzacuilti ni tlali. Nochi catli mohueyimatij xiquintlatzacuilti quen quinamiqui. 
TOTECO, ¿hasta quema tlamis tlatlanise huan mopaquilismacase nopa masehualme catli amo aqui quitlepanitaj? 
¿Hasta quema tlamis tetlaijilhuise huan mohueyimatise nopa tlajtlacolchijca masehualme? 
TOTECO, inijuanti tlahuel quintlaijiyohuiltíaj huan quintequipacholmacaj momasehualhua catli tiquinicnelía. 
Inijuanti quinmictíaj cahual toahuihua catli ehuani ipan sequinoc altepeme huan catli icnotzitzi. 
Huan quiijtohuaj para ta, TOTECO, amo tiquita, niyon amo timomaca cuenta. Quej nopa quiijtohuaj tlen ta catli tiiTeco tohuejcapan tata Jacob. 
Amojuanti catli anhuihuitique ipan ni masehualme, xijmachilica. ¿Hasta quema anquitequihuise amotlalnamiquilis para antlamachilise? 
Toteco quisencajqui tonacashua, huajca ¿anmoilhuíaj para amo quicaquis catli anquiijtohuaj? Yaya quichijchijqui toixteyol, huan ¿anmoilhuíaj para amo quiitas catli anquichihuaj? 
Yaya quintlatzacuiltía masehualme ipan nochi tlalme, ¿anmoilhuíaj amo anmechtlatzacuiltis nojquiya? Yaya quimati nochi catli pano, huan ¿anmoilhuíaj amo quimati tlaque anquichihuaj? 
TOTECO quiiximati inintlalnamiquilis masehualme huan quimati para san moilhuíaj tlamantli catli amo cuali. 
Pero cuali ininpanti nopa masehualme catli ta, TOTECO, tiquintlatzacuiltis para tiquintlalnamictis. Ya nopa quinpalehuía para quitoquilise moojhui, huan para monejnemiltise xitlahuac ipan motlanahuatilhua. 
Huan para itztose ica paquilistli quema ajsis nopa tonali catli fiero. Pero nopa masehualme catli amo quitlepanitaj TOTECO mochihuilíaj se ostotl para ipan momasiltise huan motzontlamiltise. 
Pampa TOTECO amo quintlahuelcahuas imasehualhua pampa inijuanti iaxcahua. 
Se tonal Toteco tetlajtolsencahuas xitlahuac, huan nochi catli itztoque ica ininnemilis xitlahuac, inijuanti paquise miyac. 
¿Ajqueya nechmanahuis ininmaco nopa mosisinijca masehualme? ¿Ajqueya nechmocuitlahuis tlen nopa tlaixpanoca masehualme? 
Sintla Toteco amo nechpalehuijtosquía, huajca noyoltzi ya eltosquía campa tlatzitzicatoc pampa ya nimictosquía. 
Na nitzajtzic huan niquijto: “TOTECO, xixitlaca noicxihua.” Huan yaya nechtlasojtlac huan nechmaquixti. 
NoTeco, quema san nimonejnehuilía tlen miyac tequipacholi huan san ya nopa nijmati, ta techyoltlalía. Techchihua ma nimotemachi ipan ta huan techmaca paquilistli. 
Se tlanahuatijquetl catli amo quichihua catli xitlahuac, amo tijcahuas para mosentilis ica ta. Niyon amo itztoc mohuaya catli quichihua se tlanahuatili catli amo xitlahuac. 
Pero tlanahuatiani quej nopa mosentilíaj huan mocamanalhuíaj para quenicatza quimictise se catli amo tleno quichijtoc, o se catli cuali monejnemiltía. 
Pero TOTECO yaya nechmocuitlahuía pampa tlahuel miyac chicahualistli quipiya. Ipan noTeco Dios nimotlatía huan amo tleno huelis nechajsis pampa yaya nechmanahuía. 
Huan Toteco Dios quichijtoc para ma huetzi ininpani nopa mosisinijca masehualme nochi nopa tlamantli catli amo cuali quichijtoque. Yaya quintzontlamiltis senquistoc quen quisencajtoyaj quinchihuilise sequinoc. Quena, TOTECO toDios yaya quintzontlamiltis.
