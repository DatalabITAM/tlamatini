HECHOS.
24.
Huan teipa quema panotoya macuili tonali, ajsico altepetl Cesarea Ananías nopa hueyi totajtzi. Huan nojquiya ihuaya hualajque sequinoc huehue tlacame tlen israelitame huan se tlatempalehuijquetl catli itoca Tértulo. Huan nochi yajque iixpa Gobernador Félix para quitelhuitij Pablo. 
Huan quema nopa soldados quihualicaque Pablo, Tértulo pejqui quitelhuía huan quiilhui nopa gobernador: ―Timitztlajpalohuaj, Gobernador. Timitztlascamatij miyac pampa nochi tiitztoque ica cuali. Huan ica motlalnamiquilis miyac tlamantli nemi cuali ipan totlal. 
Félix, timitztlascamatij miyac para nochi ni tlamantli catli techchihuilijtoc. Nochime campa hueli paquij ica motequi. 
Amo nijnequi nimitzcuatotonis ica miyac camanali, pero techchihuili se favor huan techtlacaquili se tlatoctzi. 
“Nelía ni tlacatl Pablo se tlaajcomanquetl. Ipan campa hueli tlali, yaya yohui campa israelitame huan tlamachtía seyoc tlamachtilistli para techxejxelos. Nojquiya quinyacana masehualme catli quineltocaj Jesús catli ejqui altepetl Nazaret. 
Huan se tonal ipan altepetl Jerusalén Pablo quinejqui quichihuas amo tlapajpactli tohueyi tiopa tiisraelitame huan yeca tiquitzquijque. Huan tijnequiyayaj tijtlajtolsencahuasquía quen quiijtohuaj totlanahuatilhua tiisraelitame, 
pero ajsico Lisias, nopa hueyi tlayacanquetl tlen miyac soldados huan quiitzqui ica miyac chicahualistli huan techquixtili. 
Huan techilhui sintla tijnequiyayaj tijtelhuise, ma tihualaca hasta nica. Huan sintla tijnequi tijtemolis nochi ni tlamantli, tijmatis xitlahuac nochi catli timitzilhuíaj.” 
Quej nopa quiijto Tértulo, huan nopa israelitame catli itztoyaj ihuaya nojquiya quiilhuijque Félix para melahuac catli quiijto. 
Huajca nopa tequitiquetl quimailhui Pablo ma camanalti huan Pablo quiijto: ―Nipaqui techcahuilijtoc nimitzilhuis catli panoc pampa nijmati ya quipiya miyac xihuitl titequichijtoc ipan ni totlal. 
Ni tlacame huelis mitzilhuise san quipiya majtlactli huan ome tonali niajsito nepa Jerusalén para nijhueyimatis Toteco. 
Huan amo aqui nechpantijtoc nitlanajnanquilía, niyon niteajcomana nepa. Amo nijchijqui catli amo cuali ipan nopa hueyi tiopamitl, niyon ipan sequinoc tiopame, niyon ipan nopa altepetl. 
Ni tlacame amo huelis mitznextilise para nelía nijchijtoc nopa tlaixpanoli ica catli nechtelhuíaj pampa amo nijchijtoc. 
Pero nimitzyolmelahua para quena, nijtoquilía ‘Nopa Ojtli’. Quinequi quiijtos nijhueyimati Toteco Dios quen Jesús tlamachti, huan ya nopa inijuanti quitocaxtía se tlaneltoquili catli amo cuali. Pero quej nopa nijhueyimati Toteco Dios catli tohuejcapan tatahua quitoquilijque. Huan nijneltoca nochi catli ijcuilijtoc ipan totlanahuatilhua tiisraelitame huan nochi catli itlajtol pannextijcahua Toteco quiijcuilojque. 
Nojquiya nijneltoca Toteco quinyolcuis masehualme catli ya mictoque, catli cuajcualme. Huan catli amo cuajcualme, huan ni tlacame nica nojquiya ya nopa quineltocaj. 
Yeca ica nochi nochicahualis nijchihua catli cuali para notlanejnehuilis ma amo nechtelhuis para nijchijtoc catli amo cuali iixpa Toteco, niyon iniixpa masehualme. 
“Quema tlanqui niquinpaxalojtinemi sequinoc tlalme para se ome xihuitl, sempa nihualajqui Jerusalén para niquinmacato tomi nopa catli teicneltzitzi nepa huan para nijmacas Toteco tlacajcahualistli. 
Huajca nijchijqui nochi catli monequi para nimotlapajpacchihuas iixpa Toteco, huan niyajqui tiopan calijtic huan nijmacayaya Toteco se tlacajcahualistli quema sequin israelitame tlen tlali Asia nechitaque. Pero amo aqui mosentiliyaya nohuaya huan amo aqui nicajcomanayaya. 
Monequi nopa israelitame catli nelía nechitaque nepa ma hualaca nica nechtelhuise sintla moilhuíaj nijchijqui catli amo cuali. 
O ma quiijtoca ni tlacame nica tlaque notlaixpanol quipantijque ipan nopa hueyi tlanechicoli tlen israelita tequitini quema nimonexti iniixpa. 
San ipan nopa tlasentililistli chicahuac niquijto: ‘Antechteilhuíaj ama pampa nijneltoca para nochi masehualme catli mictoque, teipa moyolcuise.’ ” 
Huajca Félix nimantzi quinejqui quicahuas ni cualantli hasta seyoc tonal pampa cuali quimatqui quen tlaneltocani quineltocaque Nopa Ojtli quinequi quiijtos Jesús. Huajca Félix quinilhui: ―Nijtlajtolsencahuas Pablo quema hualas Lisias nopa hueyi tlayacanquetl tlen soldados. 
Huajca Félix quiilhui nopa tlayacanquetl tlen soldados catli quimocuitlahuiyaya Pablo ma quicahuili ma quichihua catli quinequi huan para ma quipaxaloca ihuampoyohua huan ma quimacaca catli quimonequilía. 
Huan pano se ome tonali huan sempa ajsico Félix ipan altepetl Cesarea ihuaya isihua Drusila catli eliyaya se israelita. Huan quititlanqui se tlacatl ma quicuiti Pablo pampa quinequiyayaj quicaquise para tlen motemachía ipan Jesucristo. 
Huan Pablo pejqui quinilhuía quenicatza Toteco Dios quipactía san catli xitlahuac. Huan quinequi para nochi ma monahuatica para ma amo quichihuaca catli amo cuali. Huan nojquiya quinilhui para se tonal nochi masehualme monextise iixpa Toteco para yaya quintlajtolsencahuas. Huajca ica nopa camanali momajmati Félix huan quiilhui Pablo: ―Xiya ama, huan seyoc tonal quema amo tleno nijchihua, nimitznotzas techcamanalhuiqui. 
Nojquiya Félix quichiyayaya para Pablo ma quitlaxtlahui para quimajcahualtis, yeca miyacpa quinotzayaya para quicamanalhuis. 
Huan panoc ome xihuitl, huan tlanqui itequi Félix quen gobernador huan pejqui tlanahuatía Porcio Festo. Huan Félix quinequiyaya mocahuas cuali ica israelitame, huajca quicajtejqui Pablo ipan tlatzactli.
