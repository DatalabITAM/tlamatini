ÉXODO.
36.
“Huajca Bezaleel huan Aholiab huan nochi sequinoc masehualme catli TOTECO quinmacatoya tlalnamiquilistli para ni tequitl, huan quipixque ipan ininyolo para tlapalehuise, monejqui tequitise huan quisencahuase nopa yoyon tiopamitl senquistoc quen TOTECO tlanahuatijtoya.” 
Huajca Moisés quinnotzqui Bezaleel huan Aholiab, huan nochi masehualme catli TOTECO quinmacatoya tlalnamiquilistli huan catli motemacayayaj ica nochi ininyolo para tequitise, huan nochi hualajque. 
Huan Moisés quinmactili nochi tlamantli catli nopa israelitame quihualicatoyaj catli monequi quitequihuise ipan nopa yoyon tiopamitl. Huan mojmostla nopa israelitame noja quihualicayayaj nemactli para quitequihuise. 
Huajca teipa nochi inijuanti catli quisencahuayayaj tlamantli quiquetzque inintequi, 
huan quicamanalhuitoj Moisés huan quiilhuijque: “Nochi quihualicatoque tlahuel miyac tlamantli hasta conpano catli monequi para tijchihuase nochi catli TOTECO tlanahuatijtoc.” 
Huajca Moisés tlanahuati ma quinyolmelahuatij nochi israelitame ipan ininchajchaj para niyon se tlacatl o sihuatl ayacmo ma quisencahua seyoc tlamantli para nopa tiopamitl, niyon ayacmo ma tlahuajhualicaca. 
Pampa ya quipixtoyaj tlahuel miyac catli quitequihuise huan hasta miyac mocahuayaya. 
Huan nopa masehualme catli Toteco quinmacatoya tlalnamiquilistli para ma tequitica nopona, quichijque itzontzajca nopa tiopamitl. Quichijque ica 10 cortinas ica lino tlamalintli nelcanactzi huan tlaxinepaloli ica icpatl catli tlapali azultic, morado, huan chichiltic huan ipani nesiyaya inintlachiyalis nopa tequihuajme catli inintoca querubines. 
Huan sesen nopa cortinas eliyaya 12 metros huan tlajco ihuehueyaca, huan ome metro ipatlajca. 
Huan quisalojque itlamiya se cortina ica itlamiya seyoc nopa macuili cortinas para mochijqui san se cortina catli huehueyac. Huan nopa seyoc macuili cortinas nojquiya quiijtzonque para mochijqui san se cortina catli huehueyac huan nopa ome cortinas huehueyac elqui para sesen lado itzontzajca nopa tiopamitl. 
Huan ipan se iteno sesen nopa ome huehueyac cortinas catli quimojmotzquiltijtoyaj, quichihuilijque tlailpili ica iojal campa mosojsos, huan quitenyahualojque sese iojal ica icpatl azultic. 
Para nochi quichihuilijque 50 tlailpili sese ica iojal para mosojsos ipan se lado nopa ome huehueyac cortinas catli achtihui tlasaloli. Huan teipa quinchijque nopa tlailpili ipan sese para ma moixnamictica ica nopa tlailpili ipan seyoc. 
Nojquiya quichijque 50 tepos chijcoli tlen oro para quisalos nopa 50 tlailpili ipan sesen nopa ome cortinas tlasaloli. Quej nopa quisencajque para quitzacuas huejcapa ipan nopa yoyon tiopamitl. 
Nojquiya quisencajque 11 cortinas tlen nopa tetic yoyomitl catli tlaxinepaloli ica iniijhuiyo oquich chivojme para quisencajque iompa tzontzajca nopa yoyon tiopamitl. 
Huan sesen cortina ihuehueyaca eliyaya 13 metros huan tlajco, huan ome metro ipatlajca, huan nochi 11 cortinas quipixqui san se ininhuejhuexca. 
Huan quisalojque macuili cortinas san sejco huan iyoca quisalojque seyoc chicuase cortinas. 
Huan iteno se lado sesen nopa ome huehueyac cortinas quichihuilijque 50 tlailpili ica iniojales campa mosojsos. 
Nojquiya quichihuilijque 50 tepos chijcoli tlen bronce para ipan quisalose iniojales nopa tlailpili ipan se cortina ica nopa tlailpili ipan seyoc cortina para iompa tzontzajca nopa yoyon tiopamitl elis tlasaloli san sejco. 
Huan nopa expa itzontzajca quichijque ica inincuetlaxo oquich chivojme catli quipajque chichiltic, huan nopa najpa itzontzajca catli elqui más calixco quichijque ica inincuetlaxo pesojme. 
Huan para icaltepanyo nopa tiopamitl quiquejquetzque nopa huapalme tlen cuahuitl acacia. 
Huan sese huapali quipixqui nahui metros huan tlajco ihuehueyaca, huan 65 centímetros ipatlajca. 
Sese huapali quipixqui ome tepos chijcoli para quisalos ica ne sequinoc huapalme ipan sesen lado. 
Huan moquetzayayaj 20 huapalme ipan nopa caltechtli ica sur. 
Huan itzintla sesen nopa huapalme calaquiyayaj ipan ome itepos cuatetonyo tlen plata para ma moquetzas cuali. Oncayaya 40 tepos cuatetomitl, ome cuatetomitl ipan sese huapali. Huan sesen huapali quipixtoya ome tepos chijcoli para quitlatzquiltis ipan itepos cuatetonyo. 
Ipan nopa caltechtli ica norte oncayaya seyoc 20 huapalme, 
huan 20 itepos cuatetonyo. Itzintla sese huapali oncayaya ome itepos cuatetonyo. 
Iica nopa tiopamitl ipan nopa lado campa temo tonati quipixqui chicuase huapalme. 
Huan quitlalilijque seyoc huapali iica tiopamitl campa sese icalnacastla. 
Huan quen nochi más huapalme, nopa huapalme campa inincalnacastla quipixque tepos chijcoli ica tlatzintla huan ica huejcapa para quisojsos ica nopa huapalme catli inechca. 
Huajca campa temo tonati oncayaya chicueyi huapalme huan 16 tepos cuatetomitl catli ipan moquetzayayaj, ome cuatetomitl tlatzintla sesen huapali. 
Quisencajque cuaixtlapali huejhuehueyac tlen cuahuitl acacia huan quitlalijque iixpa nopa huapalme ipan nopa caltechtli catli quiyahualojque nopa tiopamitl. Quichijque macuili cuaixtlapali para iixpa nopa huapalme ipan se lado nopa tiopamitl, 
huan seyoc macuili cuaixtlapali para iixpa nopa huapalme ipan ne seyoc lado, huan seyoc macuili cuaixtlapali para iixpa huapalme catli moquetzque iica nopa tiopamitl ica campa temo tonati. 
Huan nopa cuaixtlapali catli panoc tlatlajco iixtlapal nopa huapalme catli moquetztoyaj pejque ica se icalnacastla tiopamitl huan ajsic hasta seyoc icalnacastla nopa tiopamitl. 
Huan quixajque nopa huapalme ica oro, huan nopa anillos campa calaquiyayaj nopa cuaixtlapalme elqui senquistoc oro. Nojquiya quixajque ica oro nopa cuaixtlapalme catli huejhuehueyac. 
Huan nopa cortina catli quitzajqui nopa cuarto catli más calijtic quichijque ica lino catli tlamalintli nelcanactzi huan tlaxinepaloli ica icpatl catli quipajtoyaj azultic, morado huan chichiltic. Huan ipan nopa tlaxinepaloli nesiyayaj ome tequihuajme catli inintoca querubines. 
Huan para ipan mohuiyonis nopa cortina quichijque nahui tlaquetzali ica cuahuitl acacia huan quinxajqui ica oro. Huan ipan nopa tlaquetzali quitlalijque tepos chijcoli tlen oro. Huan sesen nopa nahui tlaquetzalme moquetzayaya ipan inincuatetonyo tlen plata. 
Nojquiya quichijque se cortina para quitzacuase campa calaquise ipan yoyon tiopamitl. Quichijque ica lino nelcanactzi catli tlaxinepaloli ica icpatl azultic, morado, huan chichiltic huan quiijtzonque yejyectzi ica itlachijchihualo. 
Huan ni cortina tlatzquitoya ipan macuili tlaquetzali ica macuili tepos chijcoli. Huan inintzonatipa nopa macuili tlaquetzali huan tepos chijcoli quixajque ica oro, huan nopa macuili icuatetonyo catli ipan moquetzqui nopa tlaquetzali quichijque ica bronce.
